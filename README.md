# Food App (Next.js)

## Ссылки

- **Production (Vercel):** [https://food-app-next-alpha.vercel.app](https://food-app-next-alpha.vercel.app/)
- **Исходный репозиторий(Vite):** [https://github.com/ShiginaKatya/food-app](https://github.com/ShiginaKatya/food-app)

## Скрипты запуска

### Команды разработки

- **`yarn dev`** (или `npm run dev`)
- **`yarn build`** (или `npm run build`)
- **`yarn start`** (или `npm run start`)

### Проверка и исправление кода

- **`yarn lint`** (или `npm run lint`)
- **`yarn lint:fix`** (или `npm run lint:fix`)
